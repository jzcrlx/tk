5、[A4型题]
患者女性，29岁，因\'外阴瘙痒，白带增多有异味一周\'就诊。询问病史后，护士协助医师对该患者进行盆腔检查。 关于盆腔检查，下列叙述正确的是( )

A.对疑有盆腔内病变、腹壁肥厚、高度紧张不配合者，如盆腔检查不满意，可以在麻醉下进行盆腔检查，或行超声检查

B.检查前患者必须排空膀胱

C.患者均取膀胱截石位

D.采取双合诊或三合诊检查

E.月经期应避免检查

F.一人一单，防止交叉感染

显示答案

参考答案：A B C E F

题目解析： 双合诊用于有性生活者的盆腔检查，未婚女性一般不做双合诊，仅做直肠一腹部诊检查。如确有检查必要，应在其家属和本人同意后，用示指放入阴道叩诊；检查时应一人一单，防止交叉感染。故此题的正确答案是A、B、C、E、F，其中关键正确选项是F；此题的错误选项是D。 患者白带呈稀薄泡沫状，考虑可能为滴虫性阴道炎。取白带前应嘱咐患者避免性生活；避免阴道灌洗及阴道用药；窥阴器不应使用润滑剂；为提高检出率，要注意保存分泌物标本中滴虫的活性，不宜使用95%乙醇固定；取白带后的标本应及时送检，天气寒冷时标本要注意保暖，避免因温度过低降低滴虫活性，致使镜下无法辨认；寻找病原体毛滴虫时，采用生理盐水湿片法检出率高，显微镜下可见波状运动的滴虫，而不宜采用10%氢氧化钾溶液湿片法。故此题的正确选项是A、C；错误选项是B、D、E、F，其中关键错误选项是B。 阴道毛滴虫适宜生长的温度为25～40℃、pH为5.2～6.6的潮湿环境；月经前后阴道pH发生变化，月经后接近中性，故月经前后隐藏在腺体及阴道皱襞中的滴虫易繁殖复发；经性交直接传播是阴道毛滴虫的主要传播方式，男性感染后常无症状，易成为感染源；阴道毛滴虫感染的潜伏期为4～28天，25%～50%的患者感染初期无症状；滴虫能吞噬或消耗阴道上皮细胞内糖原，阻碍乳酸生成，降低阴道酸度以利其生存，故滴虫感染后，阴道酸度降低、pH增高(正常阴道pH3.8～4.4)；临床治疗中常依据阴道毛滴虫的这一特征，指导患者使用酸性溶液坐浴，以恢复阴道正常的、不利于毛滴虫生长的酸性环境。故此题的正确选项是B、C、D、F，其中关键正确选项是F；此题的错误选项是A、E，其中关键错误选项是A。 阴道毛滴虫感染者应选用酸性溶液进行阴道灌洗，可选择0.5%醋酸溶液或1%乳酸溶液，选择溶液的酸碱度错误可加重病情；灌洗液常规配制500～1000ml，温度为41～43℃，正常灌洗筒高度应距床沿60～70cm，对产后10天内或妇科手术2周内的患者，应采用低位灌洗，高度不超过30cm；未婚女性不得使用窥阴器，可使用导尿管进行阴道灌洗。故此题的正确选项是A、E、G，其中关键正确选项是A；此题的错误选项是B、C、D、F，其中关键错误选项是C。 阴道毛滴虫感染者在治疗期间，内裤及坐浴、洗涤用物采取煮沸5～10分钟的方法进行消毒，避免重复感染；甲硝唑可通过乳汁排泄，故哺乳期妇女在用药期间不宜哺喂婴儿；1%乳酸溶液坐浴后阴道用药效果更好，而不能使用2%碳酸氢钠液坐浴；治疗期间感染者不应进入公共游泳池；性伴侣应同时治疗；在治疗期间禁止性生活。故此题的正确选项是A、B、E，其中关键正确选项是E；此题的错误选项是C、D、F，其中关键错误选项是F。 急性阴道炎症是宫颈治疗的禁忌证，因此，该患者应在阴道炎治愈后方能治疗宫颈炎，同时治疗宫颈炎前，应首先排除宫颈癌方可治疗；宫颈炎的治疗时间，应选择在月经干净后的3～7天进行；宫颈物理治疗后，正常会出现阴道分泌物增多，甚至有大量水样排液，术后1～2周脱痂时可有少许出血，出血量不多时无需就诊；治疗后创面愈合需4～8周，在创面未完全愈合前禁止性生活、盆浴及阴道冲洗。故此题的正确选项是A、C、F，其中关键正确选项是A；此题的错误选项是B、D、E，其中关键错误选项是E。

A.手术流产适用于妊娠14周内自愿要求终止妊娠，无生殖道急性炎症、无急性传染性疾病的体温正常者

B.负压吸引术适用于妊娠10周以内者，吸引负压一般不超过500mmHg(1mmHg=0.133kPa)

C.钳刮术适用于妊娠10～14周者，应先扩张宫颈管再行手术

D.药物流产适用于妊娠70天内者，目前临床常用药物为米非司酮与米索前列醇配伍。米非司酮为黄体酮受体拮抗剂，可以阻断黄体酮活性而终止妊娠；米索前列醇为前列腺素衍化物，可以兴奋子宫肌，软化宫颈

E.并发血液疾病、血管栓塞病史、心血管疾病、青光眼、哮喘、带器妊娠者，建议采用药物流产

F.妊娠13～28周不宜继续妊娠者，可以选择乳酸依沙吖啶(利凡诺)引产或水囊引产

参考答案：A B C F

题目解析： 女性排卵受到情绪、健康状况、性生活及外界环境等多种因素的影响，可致排卵提前或推后，亦可发生额外排卵，因此，安全期避孕法不很可靠；该女性已婚，已生育1个孩子，可采取阴茎套、口服短效避孕药、宫内节育器的方法避孕。故此题的正确选项是B、C、E；此题的错误选项是A、D、F，其中关键错误选项是A。 甾体类避孕药的作用机制包括：在雌孕激素作用下，改变受精卵在输卵管内的正常运动，干扰受精卵着床；改变子宫内膜形态与功能，使子宫内膜与胚胎发育不同步，不适于受精卵着床；使宫颈黏液量减少、黏稠度增加，不利于精子穿行；影响下丘脑下部的GnRH，抑制FSH和LH的分泌，使排卵受抑制。故此题的正确选项是C。 甾体类避孕药常见不良反应有类早孕反应、不规则阴道流血，多发生于漏服避孕药后，闭经常发生于月经不规则者；雌激素可使低密度脂蛋白降低、高密度脂蛋白升高，可使凝血因子增高，单独应用雌激素、且剂量较大、时间较长时，可导致血栓性疾病的发生，目前国内使用的甾体类避孕药含雌激素30～35μg，属于低剂量甾体激素避孕药，不会增加血栓性疾病的发生率；孕激素会影响血脂蛋白代谢，故对存在心血管疾病潜在风险的妇女，不宜长期服用甾体激素避孕药；同时避孕药中的孕激素成分可减少子宫内膜癌的发病风险。因此，此题应选择的正确答案是A、C、D、F，其中首先应考虑的关键选项是D；此题不应该考虑到的选项是B、E，其中关键错误选项是E。 并发血液疾病、血管栓塞病史、心血管疾病、青光眼、哮喘、带器妊娠者，均为药物流产禁忌证，如错误选用药物流产可导致严重后果；对采取手术终止妊娠者，护理人员应协助医生确定孕妇无生殖道急性炎症、无急性传染性疾病、体温正常方可进行。故此题的正确选项是A、B、C、F，其中关键正确选项是A；此题的错误选项是D、E，其中关键错误选项是E。 一般人工流产术后出血为3～5天，随着子宫收缩及创面修复，阴道流血逐渐停止。部分妊娠组织残留，即吸宫不全，是人工流产手术的常见并发症。表现为手术后出血时间延长，出血量多.或流血停止后又再次出现多量流血。如果错误判断为正常月经复潮，则会延误治疗。故此题的正确选项是C；此题的错误选项是A、B、D、E、F，其中关键错误选项是F。 该妇女基础体温呈双相曲线，表示有排卵，可以排除妊娠、卵巢功能异常及下丘脑问题引起的闭经；患者曾行人工流产和清宫术，故应考虑其闭经的最可能原因是D；人工流产综合征(Asherman综合征)是子宫性闭经最常见的原因，多因人工流产刮宫过度或产后、流产后出血刮宫损伤子宫内膜基底层，致宫腔粘连而导致闭经。故此题的正确选项是D；此题的错误选项是A、B、C、E、F，其中关键错误选项是A。
