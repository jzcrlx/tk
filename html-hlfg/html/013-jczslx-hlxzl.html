3、[X型题]
儿童发生意外事件时，快速判断心搏骤停的主要依据有( )

A.面色苍白

B.意识突然丧失

C.抽搐

D.大动脉搏动消失

E.瞳孔散大

显示答案

参考答案：B D

题目解析：临床表现为突然昏迷,部分有一过性抽搐、呼吸停止面色灰暗或发绀、瞳孔散大和对光反射消失、大动脉(颈动脉、股动脉、肱动脉)搏动消失、听诊心音消失,如做心电图检查可见等电位线、电机械分离或心室颤动等。 心跳呼吸骤停的诊断并不困难，一般患儿突然昏迷及大血管搏动消失即可诊断;但在紧急情况下,触诊不确定有无大血管搏动亦可拟诊10秒)而不必反复触摸脉搏或听心音以免延误抢救时机。

4、[A4型题]
患儿女性，5岁，平素身体体健，因突然出现恶心、呕吐数次，昏迷伴抽搐1小时，家长抱着患儿到某医院儿科急诊室就诊。 作为急诊护士，接诊患儿时的正确处理措施是( )

A.保持沉着冷静，将患儿立即送入急诊抢救室

B.立即通知值班医生及相关人员进行抢救

C.放置床档，以防坠床

D.介绍相关的环境及规章制度

E.解开衣领、衣扣、头偏向一侧

F.填写病历相关资料

显示答案

参考答案：A B C E

题目解析： 急诊护士接到危重患者时，应直接送入抢救室，通知相关医生抢救，开通各种抢救通道，准备各种抢救仪器、放置床栏保证患儿安全等；介绍相关环境及规章制度及填写病历资料工作要等待患儿病情稳定后再实施。此题正确答案是A、B、C、E。 上述各选项均有其疾病特点：①绝大多数破伤风患者均有外伤史，伤口多先有或并发化脓性感染；②电击伤患者在电流通过的部位有电灼伤，全身表现是，轻者头晕、心悸、皮肤脸色苍白、四肢乏力等，重者可表现为昏迷、抽搐、休克、心律失常、心搏骤停；③脑血管意外又称脑卒中，以猝然昏倒、不省人事或突然发生口眼歪斜、半身不遂、舌强言蹇、智力障碍为主要特征；④中暑通常是在高温情况下才会出现，轻度表现为面色潮红、大量出汗、脉搏快速，重者可表现高热、无汗、昏迷等症状；⑤气道异物导致不完全梗阻时，会有强烈的刺激性咳嗽，完全梗阻时出现口唇发绀、意识丧失；⑥有机磷中毒患者，轻度中毒会出现恶心、呕吐、头晕、流涎、多汗、瞳孔缩小，心率减慢；中度和重度中毒患者会出现嗜睡、昏迷、抽搐、肌束颤动、双肺大量湿啰音及哮鸣音、脑水肿、呼吸衰竭等表现。案例中该患儿突然出现恶心、呕吐数次，意识丧失伴抽搐1小时，无外伤及灼伤，排除脑卒中、中暑、气道异物等的可能，目前该患儿最可能发生的情况是有机磷中毒。故此题首先应考虑的正确选项是C。 对确诊为有机磷中毒的患儿，要尽早彻底洗胃，即使超过6小时，也应积极给予彻底洗胃，常用洗胃液有生理盐水、清水、2%碳酸氢钠等。有机磷农药有多种，敌百虫、敌鼠钠中毒者禁用2%碳酸氢钠洗胃，因为敌百虫遇碱会变成更毒的敌敌畏。在紧急情况下，如一时难以确定毒物性质，应使用大量清水或生理盐水洗胃。治疗过程中要遵医嘱及时应用阿托品、解磷定等解毒药物，保持呼吸道的通畅，严密监测生命体征变化，给予氧气吸入，必要时行气管切开；切忌取仰卧位，以防呕吐物误吸造成窒息。故此题的正确答案是A、B、C、D、F；此题的错误选项是E。 生命体征平稳的患儿洗胃时要采取左侧卧位，这样可以有效减慢胃排空，延缓毒物进入十二指肠的速度，但对于生命体征不稳定、给予气管插管的患儿，洗胃时要取平卧位。洗胃时灌注量过大易引起胃扩张，使胃内压上升，加速毒素的吸收，故此题的正确选项是A、C、E、F；错误选项是B、D。 转运急危重患者时，应在转运前联系好接收患者的科室及电梯等相关辅助部门，有针对性地组织安排好相应的急救防范措施，降低患者在转运过程中的危险因素，提高抢救成功率。此题D选项，即转运时间要短，推车速度要快，明显是错误的，在转运过程中要尽量缩短转运时间，但不能一味加快推车速度，这样很容易造成碰撞、运行不稳等，导致病情波动、摔伤，甚至坠床等意外情况的发生。故此题的正确答案是A、B、C、E、F；此题的错误选项是D。 急诊护士转运患儿至病房后，要协助病房护士安置患儿，取合适体位，与病房护士双人核对患儿的姓名及腕带，交接生命体征、皮肤、管道、用药、病情变化、急诊的处理等，清点病历及护理记录单，对急诊室的处理、检查结果及尚未回报的检查结果，进行重点交接班，由转入科室填写危重患者转科交接记录本，并由双人签字确认。故此题的正确选项是A、B、C、D、E、F。 对存在意识障碍的患者，可用牙垫、开口器防止舌咬伤，做好基础护理，预防并发症的发生，设立安全护理措施防止患儿自行拔管、自伤、坠床的风险，由于患儿持续惊厥，会使用大量预防脑水肿的药物，高浓度药物在使用过程中要防止渗漏及静脉炎的发生，故此题的正确选项是A、B、C、D、E、F。 昏迷、危重及抢救的患者必须佩戴腕带，腕带字迹应清晰，须包含患者姓名、性别、年龄、科室、住院号、诊断等信息，腕带填写好，由两人核对后，共同为患者佩戴。在实施诊疗、护理活动前，应严格执行查对制度，同时核对腕带上的姓名、住院号等最少两项以上的信息。若腕带损坏、丢失或转科需要更换时，需要经两人核对后重新为患者佩戴，不可一人自行更换。故此题的正确选项是A、B、E、F；此题的错误选项是C、D。 儿童是容易发生意外事件的群体，儿童好奇心强，对任何事物都有探知欲望，且喜欢模仿，易接触危险物品和模仿从电视、电影或连环画中看到的危险动作，加上年幼无知，缺乏生活经验，对意外事故缺乏警惕性与分辨力。家长要加强对儿童安全的管理及监护；重视对儿童的安全教育及训练；将危险药物、器具等物品放在儿童不易拿取的地方；在儿童发育时期，也不可为了避免意外事件的发生而限制其活动的范围。故此题的正确选项是B、C、D、E、F；此题的错误选项是A。

6、[A4型题]
患儿女性，5岁，平素身体体健，因突然出现恶心、呕吐数次，昏迷伴抽搐1小时，家长抱着患儿到某医院儿科急诊室就诊。 提示：据家长表诉，家中院子里装有机磷杀虫药的饮料瓶，有被人打开过的痕迹。患儿急查血，检验结果显示：白细胞11.4×10/L、血红蛋白125g/L、血小板总数156×10/L，胆碱酯酶为990U/L，诊断为有机磷中毒。针对有机磷中毒的患儿，正确的护理措施是( )

A.遵医嘱给予阿托品及解磷定

B.保持呼吸道通畅

C.清水或生理盐水洗胃

D.监测生命体征

E.取仰卧位，保持呼吸通畅

F.必要时气管切开

题目解析： 急诊护士接到危重患者时，应直接送入抢救室，通知相关医生抢救，开通各种抢救通道，准备各种抢救仪器、放置床栏保证患儿安全等；介绍相关环境及规章制度及填写病历资料工作要等待患儿病情稳定后再实施。此题正确答案是A、B、C、E。 上述各选项均有其疾病特点：①绝大多数破伤风患者均有外伤史，伤口多先有或并发化脓性感染；②电击伤患者在电流通过的部位有电灼伤，全身表现是，轻者头晕、心悸、皮肤脸色苍白、四肢乏力等，重者可表现为昏迷、抽搐、休克、心律失常、心搏骤停；③脑血管意外又称脑卒中，以猝然昏倒、不省人事或突然发生口眼歪斜、半身不遂、舌强言蹇、智力障碍为主要特征；④中暑通常是在高温情况下才会出现，轻度表现为面色潮红、大量出汗、脉搏快速，重者可表现高热、无汗、昏迷等症状；⑤气道异物导致不完全梗阻时，会有强烈的刺激性咳嗽，完全梗阻时出现口唇发绀、意识丧失；⑥有机磷中毒患者，轻度中毒会出现恶心、呕吐、头晕、流涎、多汗、瞳孔缩小，心率减慢；中度和重度中毒患者会出现嗜睡、昏迷、抽搐、肌束颤动、双肺大量湿啰音及哮鸣音、脑水肿、呼吸衰竭等表现。案例中该患儿突然出现恶心、呕吐数次，意识丧失伴抽搐1小时，无外伤及灼伤，排除脑卒中、中暑、气道异物等的可能，目前该患儿最可能发生的情况是有机磷中毒。故此题首先应考虑的正确选项是C。 对确诊为有机磷中毒的患儿，要尽早彻底洗胃，即使超过6小时，也应积极给予彻底洗胃，常用洗胃液有生理盐水、清水、2%碳酸氢钠等。有机磷农药有多种，敌百虫、敌鼠钠中毒者禁用2%碳酸氢钠洗胃，因为敌百虫遇碱会变成更毒的敌敌畏。在紧急情况下，如一时难以确定毒物性质，应使用大量清水或生理盐水洗胃。治疗过程中要遵医嘱及时应用阿托品、解磷定等解毒药物，保持呼吸道的通畅，严密监测生命体征变化，给予氧气吸入，必要时行气管切开；切忌取仰卧位，以防呕吐物误吸造成窒息。故此题的正确答案是A、B、C、D、F；此题的错误选项是E。 生命体征平稳的患儿洗胃时要采取左侧卧位，这样可以有效减慢胃排空，延缓毒物进入十二指肠的速度，但对于生命体征不稳定、给予气管插管的患儿，洗胃时要取平卧位。洗胃时灌注量过大易引起胃扩张，使胃内压上升，加速毒素的吸收，故此题的正确选项是A、C、E、F；错误选项是B、D。 转运急危重患者时，应在转运前联系好接收患者的科室及电梯等相关辅助部门，有针对性地组织安排好相应的急救防范措施，降低患者在转运过程中的危险因素，提高抢救成功率。此题D选项，即转运时间要短，推车速度要快，明显是错误的，在转运过程中要尽量缩短转运时间，但不能一味加快推车速度，这样很容易造成碰撞、运行不稳等，导致病情波动、摔伤，甚至坠床等意外情况的发生。故此题的正确答案是A、B、C、E、F；此题的错误选项是D。 急诊护士转运患儿至病房后，要协助病房护士安置患儿，取合适体位，与病房护士双人核对患儿的姓名及腕带，交接生命体征、皮肤、管道、用药、病情变化、急诊的处理等，清点病历及护理记录单，对急诊室的处理、检查结果及尚未回报的检查结果，进行重点交接班，由转入科室填写危重患者转科交接记录本，并由双人签字确认。故此题的正确选项是A、B、C、D、E、F。 对存在意识障碍的患者，可用牙垫、开口器防止舌咬伤，做好基础护理，预防并发症的发生，设立安全护理措施防止患儿自行拔管、自伤、坠床的风险，由于患儿持续惊厥，会使用大量预防脑水肿的药物，高浓度药物在使用过程中要防止渗漏及静脉炎的发生，故此题的正确选项是A、B、C、D、E、F。 昏迷、危重及抢救的患者必须佩戴腕带，腕带字迹应清晰，须包含患者姓名、性别、年龄、科室、住院号、诊断等信息，腕带填写好，由两人核对后，共同为患者佩戴。在实施诊疗、护理活动前，应严格执行查对制度，同时核对腕带上的姓名、住院号等最少两项以上的信息。若腕带损坏、丢失或转科需要更换时，需要经两人核对后重新为患者佩戴，不可一人自行更换。故此题的正确选项是A、B、E、F；此题的错误选项是C、D。 儿童是容易发生意外事件的群体，儿童好奇心强，对任何事物都有探知欲望，且喜欢模仿，易接触危险物品和模仿从电视、电影或连环画中看到的危险动作，加上年幼无知，缺乏生活经验，对意外事故缺乏警惕性与分辨力。家长要加强对儿童安全的管理及监护；重视对儿童的安全教育及训练；将危险药物、器具等物品放在儿童不易拿取的地方；在儿童发育时期，也不可为了避免意外事件的发生而限制其活动的范围。故此题的正确选项是B、C、D、E、F；此题的错误选项是A。
