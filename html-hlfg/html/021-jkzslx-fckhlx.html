9、[X型题]
关于妊娠合并急性病毒性肝炎正确的是

A.肝炎患者原则上不宜妊娠

B.分娩期及产褥期注意防止母婴传播及产后出血

C.应用抗生素预防产褥感染

D.新生儿出生后6小时和一个月时各肌注1mlHBIG

E.患急性肝炎痊愈后即可妊娠

显示答案

参考答案：A B C

题目解析：乙型肝炎是最常见的肝炎，主要经血液传播，但母婴传播是其重要的途径，我国高达50%的慢性HBV感染者是经母婴传播造成的。乙型病毒性肝炎在妊娠期更容易进展为重型肝炎，是我国孕产妇死亡的主要原因之一。感染HBV的育龄女性在妊娠前应行肝功能、血清 HBV DNA检测以及肝脏B型超声检查。最佳的受孕时机是肝功能正常、血清 HBV DNA低水平、肝脏B型超声无特殊改变。已患肝炎的育龄妇女应做好避孕。患急性肝炎者应于痊愈后半年,最好2年后在医师指导下妊娠。故A选项正确，E选错错误。 肝炎患者会影响凝血功能，故在妊娠期要注意观察是否出血，产褥期预防产后出血。妊娠中后期以及产后遵医嘱用药，预防母婴传播。故B正确。 产褥期给予对肝脏损害较小的抗生素预防感染。故C正确。 HBsAg阳性母亲的新生儿应在出生后24小时内尽早(最好在出生后12小时)注射乙型肝炎免疫球蛋白（HBIG），在1个月和6个月时分别接种第2和第3针乙型肝炎疫苗。故D不正确。

19、[X型题]
有关贫血孕妇的叙述，正确的是

A.缺铁性贫血补充铁剂以静脉补充为主

B.血红蛋白<60g/L缺铁性贫血孕妇早期可以少量多次输血

C.长期偏食或挑食可引起巨幼红细胞性贫血

D.贫血易引起流产、胎儿宫内发育迟缓等

E.缺铁性贫血孕妇应告知其铁剂应饭前口服

显示答案

参考答案：B C D

题目解析：A缺铁性贫血以口服铁剂为主。 B重度贫血的产妇于临产后配血备用，遵循少量多次的原则。 C妊娠期的时候由于恶心，呕吐，食欲下降，或者是长期偏食或挑食，有可能出现叶酸的摄入量减少；或者是孕妇存在着一些胃肠道疾病的时候，比如说慢性萎缩性胃炎等，有可能会导致维生素B12的吸收障碍，患者有可能会出现叶酸和维生素B12的缺乏，从而有可能会导致巨幼红细胞性贫血的发生。 D若孕妇缺铁严重时,会影响骨髓造血功能致重度贫血,则缺乏胎儿生长发育所需的营养物质和胎盘养分,可造成胎儿生长受限、胎儿宫内窘迫、早产、死胎或死产等不良后果。 E铁剂对胃黏膜有刺激作用，应该饭中或饭后服用。
