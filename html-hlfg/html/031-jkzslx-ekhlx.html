10、[X型题]
化脓性脑膜炎的常见护理诊断有

A.体温异常

B.营养失调

C.有窒息的危险

D.有体液不足的危险

E.有感染的危险

显示答案

参考答案：A B C D

题目解析：体温过高 与细菌感染有关。 潜在并发症：颅内压增高 与颅内感染、水电解质紊乱、硬脑膜下积液等有关。 有窒息的危险 与颅内压增高引起频繁呕吐，小儿呛咳反射弱等有关。 有体液不足的危险 与频繁呕吐、高热出汗有关。 有受伤的危险 与惊厥发作有关。 营养失调：低于机体需要量 与摄入不足、机体消耗增多有关。 焦虑（家长） 与疾病预后不良有关。 化脓性脑膜炎本就是各种化脓性细菌感染引起的急性脑膜炎症，E错。

17、[X型题]
病毒性脑膜炎的主要临床表现包括

A.发热

B.头痛

C.颈项强直

D.颈背疼痛

E.意识障碍

显示答案

参考答案：A B C D

题目解析：病毒性脑膜炎：多先有上呼吸道或消化道感染病史，表现为发热、恶性、呕吐。继而婴儿出现烦躁不安，易被激惹；年长儿表现头痛、颈背疼痛，脑膜刺激征阳性。很少发生严重意识障碍和惊厥，无局限性神经系统体征。

18、[X型题]
脑瘫的临床表现包括

A.癫痫

B.智力低下

C.语言功能障碍

D.视觉功能障碍

E.中枢性随意肌功能受累

显示答案

参考答案：A B C D E

题目解析：脑性瘫痪的常见症状：1.运动障碍：是脑瘫患儿的最基本表现，其特征是运动发育落后和瘫痪肢体主动运动减少，肌张力、姿势及神经反射异常。2.伴随症状：约半数脑瘫患儿伴有智力低下，听力、语言、视力障碍，认知和心理行为异常及癫痫等一系列发育异常的症状。
