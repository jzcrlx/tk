1、[A4型题]
三十二岁女性，近2年来月经量增多，经期延长，无腹痛，妇检宫颈轻度糜烂，子宫前位妊娠2个月大小，表面结节感活动，无明显压痛，双附件阴性。 最有可能的诊断是

A.宫颈炎

B.子宫内膜癌

C.子宫肌瘤

D.月经失调

E.附件炎

显示答案

参考答案：A C

题目解析：慢性子宫颈炎、子宫颈上皮内瘤变、甚至早期子宫颈癌都可呈现子宫糜烂性改变。宫颈糜烂是慢性炎症刺激引起的，是宫颈炎的一种表现形式。经量增多及经期延长、子宫前位妊娠2个月大小，表面结节感活动的表现符合子宫肌瘤的诊断。所以患者最可能的诊断是宫颈炎合并子宫肌瘤。

2、[A4型题]
三十二岁女性，近2年来月经量增多，经期延长，无腹痛，妇检宫颈轻度糜烂，子宫前位妊娠2个月大小，表面结节感活动，无明显压痛，双附件阴性。 患者接受手术治疗时护理指导正确的是

A.讲解有关的临床表现

B.讲解本病是女性中最少见得良性肿瘤

C.讲解手术的治疗意义

D.耐心解答有关的问题

E.手术中不会出现意外

显示答案

参考答案：A C D

题目解析：本病是较常见的良性肿瘤。任何一个手术都不能百分百保证没有意外。

A.手术治疗

B.化疗

C.激素治疗

显示答案

参考答案：A C E
