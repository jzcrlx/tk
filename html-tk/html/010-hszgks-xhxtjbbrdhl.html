1/6. 心包积液的临床表现不包括( ) 【单选题】

A.心脏搏动减弱

B.心音低钝、遥远

C.左肩胛下区叩诊浊音

D.出现呼吸困难

E.心包摩擦音

显示答案

参考答案：E

解释与评论:

1、纤维蛋白性心包炎特有心包摩擦音

2、纤维蛋白性心包炎所特有的

3、心包摩擦音：是纤维蛋白性心包炎的典型体征

2/6. 属于缩窄性心包炎最常见的病因是( ) 【单选题】

A.结核性

B.化脓性

C.创伤性

D.肿瘤

E.放射线

显示答案

参考答案：A
解释与评论:

3/6. 患者男性，40岁。患急性心包炎、心包积液2月余。近几日出现咳嗽、活动后气促。有心绞痛样胸痛。体检：有颈静脉怒张、肝大、腹水、下肢水肿、心率增快，可见Kussmanl征。考虑诊断为 【单选题】

A.急性心包炎

B.缩窄性心包炎

C.亚急性心包炎

D.渗出性心包炎

E.纤维蛋白性心包炎

显示答案

参考答案：B
解释与评论:

4/6. 患者男性。40岁，一月前诊断为急性心包炎，近两周呼吸困难严重。心率加快。查体发现患者有奇脉。奇脉的表现是 【单选题】

A.脉搏搏动呈吸气性显著减弱，呼气时消失

B.脉搏搏动呈吸气性显著消失，呼气时减弱

C.脉搏搏动呈呼气性显著减弱或消失,吸气时减弱或有停顿

D.脉搏搏动呈呼气性显著减弱或消失，吸气时又复原

E.脉搏搏动呈吸气性显著减弱或消失，呼气时又复原

显示答案

参考答案：E

解释与评论:

1、脉：又称吸停脉，是指平静吸气时脉搏明显减弱甚至消失。是心包填塞的重要体征之一。因心包原因所致心脏舒张严重受限，回心血量减少，左心排血量减少所致

D.X线检查

E.超声心动图
